# 🛡️ CIFAR Fraud Detection using Machine Learning

## 📖 Project Overview

This project focuses on detecting fraudulent patterns using Machine Learning techniques.  
The notebook demonstrates a complete end-to-end ML workflow including:

- Data preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering  
- Model training  
- Model evaluation  

The goal is to build a classification model capable of accurately identifying fraudulent cases.
